# سیستم تشخیص و شناسایی افراد
# Person Recognition System

یک سیستم جامع و حرفه‌ای برای تشخیص، شناسایی و ثبت اطلاعات افراد با استفاده از فناوری‌های هوش مصنوعی.

## 🎯 ویژگی‌های اصلی

✅ **تشخیص خودکار افراد** - با استفاده از YOLOv8 (نسخه‌های n و s)  
✅ **تشخیص چهره** - شناسایی دقیق چهره‌ها در تصویر  
✅ **تشخیص احساسات** - شناسایی 7 حالت احساسی مختلف  
✅ **پایگاه داده امن** - ذخیره اطلاعات کاربران و لاگ دسترسی‌ها  
✅ **رابط کاربری حرفه‌ای** - طراحی مدرن و کاربرپذیر  
✅ **ثبت کاربر جدید** - اضافه کردن آسان و سریع کاربران  
✅ **لاگ دسترسی** - رصد و ثبت تمام دسترسی‌ها  

## 📋 نیازمندی‌ها

```bash
Python 3.8+
```

### کتابخانه‌های مورد نیاز:
- streamlit - رابط کاربری
- opencv-python - پردازش تصویر
- ultralytics - مدل‌های YOLO
- torch - محرک یادگیری عمیق
- deepface - تشخیص چهره و احساسات
- pillow - کار با تصاویر
- numpy - محاسبات عددی

## 🚀 شروع سریع

### 1️⃣ نصب وابستگی‌ها

```bash
pip install -r requirements.txt
```

### 2️⃣ آماده‌سازی مدل‌ها

مدل‌های YOLO را در پوشه `models/` قرار دهید:
- `yolov8n.pt` یا `yolov8s.pt` - برای تشخیص افراد
- `license_plate.pt` - برای تشخیص پلاک (اختیاری)
- `best.pt` - برای مدل‌های سفارشی (اختیاری)

### 3️⃣ اجرای برنامه

```bash
streamlit run app.py
```

سپس مرورگر خود را بازکنید و به `http://localhost:8501` بروید.

## 📊 ساختار پایگاه داده

### جدول `users`
```sql
- id: شناسه
- name: نام
- family_name: نام خانوادگی
- age: سن
- job: شغل
- phone: شماره تماس
- email: ایمیل
- image_path: مسیر عکس
- created_at: تاریخ ایجاد
- updated_at: تاریخ به‌روزرسانی
```

### جدول `access_logs`
```sql
- id: شناسه
- user_id: شناسة کاربر (NULL برای غیرمجاز)
- status: وضعیت (✅ مجاز / ❌ غیرمجاز)
- emotion: احساس شخص
- timestamp: تاریخ و زمان
- image_path: مسیر عکس
- confidence: سطح اطمینان
```

## 🔧 تنظیمات

# حداقل اطمینان برای تشخیص

# مسیرهای فایل‌ها

## 🐛 عیب‌یابی

### مشکل: مدل YOLO بارگذاری نمی‌شود
**حل:** مدل‌های مورد نیاز را از [Ultralytics](https://github.com/ultralytics/yolov8) دانلود کنید.

### مشکل: تشخیص احساسات کار نمی‌کند
**حل:** deepface را دوباره نصب کنید:
```bash
pip install --upgrade deepface
```

### مشکل: خطای CUDA
**حل:** اگر GPU ندارید، CPU استفاده خواهد شد (کندتر).
